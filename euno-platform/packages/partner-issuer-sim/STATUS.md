# STATUS — partner-issuer-sim

**Status: GA (v1.0.0) — Production-ready for Stage 5 enterprise deployments.**

This package simulates a partner-organization capability issuer. It is used
by the integration-test harness and as the reference implementation for the
DID-based cross-org trust chain.
