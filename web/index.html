📄 euno.policy.yaml 
# euno.policy.yaml — AgentCapabilityManifest
# Validated by: npx @euno/mcp validate ./euno.policy.yaml

agentId: filesystem-agent
name: Filesystem Agent
version: 1.0.0

requiredCapabilities:
  - resource: "tool://read_file"
    actions: [call]
    conditions:
      - type: allowedExtensions
        extensions: [".csv", ".json", ".txt"]
      - type: maxCalls
        count: 50
        windowSeconds: 60

  - resource: "tool://list_directory"
    actions: [call]
What you get

Everything you need to ship agents safely

Euno's MCP proxy sits between your AI host and upstream server — enforcing policy without touching your agent code.

🛡️

Policy enforcement

YAML-driven rules block disallowed tool calls at the proxy layer — before the upstream server sees them. Works with any MCP server.

📋

OCSF audit log

Every tool call is recorded in a cryptographically signed, tamper-evident OCSF audit log at ~/.euno/audit.jsonl.

🔌

Drop-in for Claude & Cursor

Wrap any existing MCP server with a one-line config change. Works with Claude Desktop, Cursor, LangChain.js, and any MCP host.

🪪

Agent identity (Stage 3)

Cryptographically verifiable JWT capability tokens per agent — scoped to role and permissions, issued by a hosted gateway. Coming in Stage 3.

How it works

Proxy sits between host and server

The MCP proxy is transparent to both sides — it speaks the MCP protocol on both ends and evaluates policy in the middle.

📝

Write policy

Define allowed tools, argument constraints, and deny-all fallback in a single euno.policy.yaml.

🔀

Wrap your server

Pass npx @euno/mcp proxy as the command in your MCP host config. No agent or server changes needed.

✅

Enforce at runtime

Every tools/call is evaluated against the policy before it reaches the upstream.

🔍

Inspect the log

Run euno audit tail to stream a live feed of every decision with full context.

Quick start

Zero-infra. Ships in 60 seconds.

No sign-up. No cloud account. Runs entirely on your machine.

1 — run the proxy
$ npx -y @euno/mcp proxy --policy ./euno.policy.yaml \
-- npx -y @modelcontextprotocol/server-filesystem /data
2 — tail the audit log
$ euno audit tail
